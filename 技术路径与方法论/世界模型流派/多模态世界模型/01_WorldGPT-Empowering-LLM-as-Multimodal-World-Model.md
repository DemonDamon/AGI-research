# WorldGPT: Empowering LLM as Multimodal World Model

**类型**: 论文 (Paper)

**链接**: [https://arxiv.org/abs/2404.18202](https://arxiv.org/abs/2404.18202)

## 摘要

Introduces WorldGPT, a generalist world model built upon Multimodal Large Language Model (MLLM). It acquires an understanding of the world and is evaluated on a new multimodal state transition dataset, WorldNet.
