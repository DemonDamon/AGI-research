# A systematic evaluation of large language models of code

**类型**: 论文 (Paper)

**链接**: [https://dl.acm.org/doi/abs/10.1145/3520312.3534862](https://dl.acm.org/doi/abs/10.1145/3520312.3534862)

## 摘要

一篇系统性评估代码大型语言模型（LLMs）的论文，对比了包括 Codex 在内的多个模型，并介绍了开源的 PolyCoder 模型。它是理解代码 LLM 基础研究的重要文献。
