# Vibe coding: programming through conversation with artificial intelligence

**类型**: 论文 (Paper)

**链接**: [https://arxiv.org/abs/2506.23253](https://arxiv.org/abs/2506.23253)

## 摘要

The first empirical study of "vibe coding," an emerging programming paradigm where developers interact with code-generating LLMs rather than writing code directly. It investigates programmer goals, workflows, prompting techniques, and challenges.
